# CampusFoodCourt

Campus Food Court is a full-stack Spring Boot web application developed to digitalize and streamline food ordering, table booking, and food service management within a college campus environment. The system provides a smart and efficient dining experience for students, teachers, and food court administrators through role-based access and real-time order handling.

## Project Overview

The application automates traditional campus food court operations by enabling users to:
- Browse food menus online
- Pre-order meals
- Reserve tables
- Track order status in real-time
- Manage food services through an admin dashboard

The system reduces waiting time, improves operational efficiency, and enhances the overall campus dining experience.


## Objectives

- Minimize long queues during peak hours
- Simplify food ordering and table booking
- Provide secure role-based access control
- Improve order management and service efficiency
- Create a responsive and user-friendly dining platform


## User Roles

### Student
- Register and login securely
- Browse available menu items
- Pre-order food online
- Book tables for specific time slots
- Track order status
- Submit ratings and feedback

### Teacher
- Order food online
- Schedule desk delivery
- View order updates and notifications

### Admin
- Manage food menu items
- Add, edit, and remove food entries
- Manage orders and bookings
- Monitor peak-hour activity
- Generate reports and analytics

## How to Run the Project
1. Clone the repository
2. Open the project in Spring Tool Suite (STS)
3. Configure MySQL database in `application.properties`
4. Run the application as Spring Boot App
5. Open browser and visit:  http://localhost:8080
